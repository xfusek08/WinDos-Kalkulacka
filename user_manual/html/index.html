Zašedlý text tlačítek

V případě, že některé z tlačítek nelze stisknout, tak je text tlačítka šedý. Takováto tlačítka nejsou povolena pro stisk. Jedná se o tlačítka jiných soustav, než ve které je kalkulačka aktuálně přepnutá anebo o tlačítka, která nelze v dané situaci stisknout. Příkladem takové situace je například nemožnost stisknutí uzavírací závorky předtím, než existuje stejný počet závorek otevíracích.
